# Netflix Clone

A responsive Netflix-inspired web application built with **HTML** and **CSS**. This project replicates the look and feel of Netflix's user interface with a modern, sleek design.

## 🌐 Live Site

**👉 [View Live Demo](https://aman-raj048.github.io/Netflix-Clone/)**

## 🎬 Features

- **Responsive Design**: Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and intuitive user interface inspired by Netflix's design
- **CSS Grid & Flexbox**: Utilizes modern CSS layout techniques for flexible component arrangement
- **Smooth Animations**: Subtle hover effects and transitions for enhanced user experience
- **Movie Showcase**: Display movies and TV shows in an organized grid layout
- **Navigation Bar**: Sticky header with navigation links and branding

## 🛠️ Tech Stack

- **HTML5** (60.5%): Semantic markup for structured content
- **CSS3** (39.5%): Advanced styling with animations, gradients, and responsive design

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies or installations required

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Aman-raj048/Netflix-Clone.git
   cd Netflix-Clone
   ```

2. **Open the project**
   - Simply double-click `index.html` to open in your default browser, or
   - Right-click `index.html` → Open with → Select your preferred browser

3. **View the project**
   - The Netflix clone interface will load in your browser
   - Browse through the movie/show listings and explore the UI

## 💡 Usage

- **Hover over content**: Interactive hover effects reveal additional information
- **Navigation**: Use the navigation bar to switch between sections
- **Responsive viewing**: Resize your browser window to see the responsive design in action

## 🎨 Customization

To customize the project:

1. **Modify the content** in `index.html` to add your own movies or shows
2. **Update colors and fonts** in `styles.css` to match your preferred theme
3. **Add more images** to the `images/` folder and update references in HTML
4. **Adjust breakpoints** in CSS media queries for different screen sizes

## 📱 Responsive Breakpoints

The project includes responsive design for:
- **Desktop**: 1024px and above
- **Tablet**: 768px to 1023px
- **Mobile**: Below 768px
